# multiACE Web contract fixtures

These payloads mirror the state contract emitted by
[`decay71/multiACE`](https://github.com/decay71/multiACE) at commit
`7c2a5e0ee6ed0b72479211efd80cb2bceb70967c`.

Field values are synthetic and cover multiple units, both routing modes, RFID and manual metadata, loaded toolheads, and dryer telemetry. They
are not exports from a user's printer.
